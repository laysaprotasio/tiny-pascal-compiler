# TinyPascal Analyzer

Projeto em Node.js que implementa um analisador léxico e sintático para a linguagem **TinyPascal** 

## Requisitos

* Node.js 

## Instalação

```bash
git clone 
cd tiny-pascal-compiler
npm install
```

## Estrutura

```
src/lexer   – scanner
examples    – programas de exemplo
```
